\begin{problem}{8.13}
Define $A_{\text{LBA}} = \{\langle M,w \rangle \ | \ M \text{ is an LBA that accepts input } w\}$. Show that $A_{\text{LBA}}$ is PSPACE-complete.
\end{problem}

\begin{proof}
To show that $A_{\text{LBA}}$ is PSPACE-complete, we must show that it is in PSPACE and that all PSPACE problems are polynomial time reducible to it. \\

To show that $A_{\text{LBA}}$ is PSPACE-complete, we can construct a PSPACE decider for $A_{\text{LBA}}$ by using the algorithm given in Theorem 5.9\footnote{An LBA operates in exactly $n$ space. This solution applies to both DLBA and NLBA due to Savitch's Theorem.}.

Part (b) Notes:

Show GG is polynomial time reducible to $A_{\text{LBA}}$.
Rewrite PSPACE decider for GG to use a stack.
The mapping $f$ takes a directed graph and outputs an LBA, graph and enough blank space for stack.
Size of stack is same as number of nodes in graph.

\end{proof}
