\begin{problem}{7.54}
In a directed graph, the \textbf{\textit{indegree}} of a node is the number of incoming edges and the \textbf{\textit{outdegree}} is the number of outgoing edges. Show that the following problem is NP-complete. Given an undirected graph $G$ and a designated subset $C$ of $G$'s nodes, is it possible to convert $G$ to a directed graph by assigning directions to each of its edges so that every node in $C$ has indegree 0 or outdegree 0, and every other node in $G$ has indegree at least 1?
\end{problem}

\begin{proof}
Let
\begin{align*}
    U = \{\langle G, C \rangle \ | \ &G \text{ is an undirected graph, } C \text{ is a designated subset of } G \text{'s nodes,} \\
    & \text{and it is possible to convert $G$ to a directed graph by assigning directions} \\
    & \text{to each of its edges so that every node in $C$ has indegree 0 or outdegree 0,} \\
    & \text{and every other node in $G$ has indegree at least 1}\}.
\end{align*}

To show that $U$ is NP-complete, we must show that it is in NP and that all NP-problems are polynomial time reducible to it. The first part is easy; a certificate is simply the directed graph. To prove the second part, we show that $3SAT$ is polynomial time reducible to $U$. The reduction converts a Boolean formula $\phi$ in 3CNF into an undirected graph $G$ and a designated subset $C$ of $G$'s nodes, so that $\phi$ is satisfiable, iff it is possible to convert $G$ to a directed graph by assigning directions to each of its edges so that every node in $C$ has indegree 0 or outdegree 0, and every other node in $G$ has indegree at least 1. \\

Let $\phi$ be any Boolean formula in 3CNF containing $m$ clauses:
\[
\phi = (a_1 \vee b_1 \vee c_1) \ \wedge (a_2 \vee b_2 \vee c_2) \ \wedge \cdots \wedge (a_m \vee b_m \vee c_m).
\]
where each $a$, $b$ and $c$ is a literal $x_i$ or $\overline{x_i}$, and $x_1, x_2, \ldots , x_n$ are the $n$ variables of $\phi$. Now we show how to convert $\phi$ into $G$ and $C$. The graph contains gadgets that mimic the variables and clauses of the formula. The variable gadget for each variable $x_i$ is a pair of adjacent nodes, labeled $x_i$ and $\overline{x_i}$. The graph $G$ contains a node labeled $Q_i$ for every clause $Q_i$ in $\phi$. Every clause node is connected with literal nodes $a_i$, $b_i$ and $c_i$. The set $C$ contains all nodes labeled $x_i$ and $\overline{x_i}$.

\begin{center}
\includegraphics[scale=1.0]{Figures/Problem7.54.pdf} \\
The graph $G$ for unsatisfiable $\phi = (x_1 \vee x_1 \vee x_1) \ \wedge (x_2 \vee x_2 \vee x_2) \ \wedge \ (\overline{x_1} \vee \overline{x_1} \vee \overline{x_2})$. \\
$C = \{x_1, \overline{x_1}, x_2, \overline{x_2}\}$.
\end{center}

Suppose $\phi$ has a satisfiable assignment. \\

Suppose it is possible to convert $G$ to a directed graph by assigning directions to each of its edges so that every node in $C$ has indegree 0 or outdegree 0, and every other node in $G$ has indegree at least 1. Then, the indegrees and outdegrees of nodes in $C$ give satisfying assignment to the variables of $\phi$. If indegree of a node $x_i$ is 0, then the variable $x$ is assigned true, otherwise false. 

\end{proof}
