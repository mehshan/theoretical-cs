\begin{problem}{2.25}
For any language $A$, let $SUFFIX(A) = \{v \ | \ uv \in A \ for \ some \ string \ u\}$. Show that the class of context-free languages is closed under the $SUFFIX$ operation.
\end{problem}

Characteristics of PDA that accepts Suffix(A):
1. start state is also an accept state
2. direct (e,e->$)transition to every other state
3. stack operations are in play

Solve it by manipulating grammer of a... subsequently remove symbols
from left in each role... avoid mixups
